# Comparações

Servem para comparar dois valores e/ou variáveis.

- Igual - `==`
- Diferente - `!=`
- Menor - `<`
- Menor igual - `<=`
- Maior - `>`
- Maior igual - `>=`

Uma comparação sempre retornará verdadeiro ou falso:

```javascript
1 > 2
1 == 1
"A" == "A"
"A" < "B"
```

Para variáveis:

```javascript
const idade = 16
const limite = 18

idade >= limite
```
